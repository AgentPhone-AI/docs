## March 2026

### MCP Server

- Added `make_conversation_call` tool for autonomous AI-driven calls
- Added `list_conversations` and `get_conversation` tools for SMS threads
- Support for Claude Code, Cursor, and Windsurf integrations

### API

- Conversation calls with built-in LLM — no webhook required
- SMS conversation threading with full message history
- Webhook signature verification with HMAC-SHA256
- US and CA phone number provisioning
