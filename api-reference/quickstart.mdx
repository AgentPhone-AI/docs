Get up and running in under 2 minutes:

### Step 1: Get your API key

<Card title="Create a project" icon="key" href="https://agentphone.to">
  Sign up at agentphone.to and grab your API key from the dashboard.
</Card>

### Step 2: Create an agent

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.agentphone.to/v1/agents \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"name": "Support Bot", "description": "Handles customer support"}'
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    import { AgentPhone } from 'agentphone';

    const client = new AgentPhone({ apiKey: process.env.AGENTPHONE_API_KEY });

    const agent = await client.agents.create({
      name: 'Support Bot',
      description: 'Handles customer support',
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from agentphone import AgentPhone
    import os

    client = AgentPhone(api_key=os.environ['AGENTPHONE_API_KEY'])

    agent = client.agents.create(
        name='Support Bot',
        description='Handles customer support',
    )
    ```
  </Tab>
</Tabs>

<CodeGroup>
```json Response
{
  "id": "agent_123",
  "name": "Support Bot",
  "description": "Handles customer support",
  "createdAt": "2025-12-03T10:00:00Z",
  "numbers": []
}
```
</CodeGroup>

### Step 3: Give it a phone number

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.agentphone.to/v1/numbers \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"country": "US", "agentId": "agent_123"}'
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    const number = await client.numbers.buy({
      country: 'US',
      agentId: agent.id,
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    number = client.numbers.buy(country='US', agent_id=agent.id)
    ```
  </Tab>
</Tabs>

<CodeGroup>
```json Response
{
  "id": "clx123abc",
  "phoneNumber": "+18571234567",
  "country": "US",
  "status": "active",
  "agentId": "agent_123",
  "createdAt": "2025-12-03T10:00:00Z"
}
```
</CodeGroup>

### Step 4: Connect your agent brain

Register a webhook to receive messages and voice transcripts:

<Tabs>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.agentphone.to/v1/webhooks \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"url": "https://your-server.com/webhook"}'
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    const webhook = await client.webhooks.set({
      url: 'https://your-server.com/webhook',
    });
    ```
  </Tab>
  <Tab title="Python">
    ```python
    webhook = client.webhooks.set(url='https://your-server.com/webhook')
    ```
  </Tab>
</Tabs>

<CodeGroup>
```json Response
{
  "id": "wh_abc123",
  "url": "https://your-server.com/webhook",
  "secret": "whsec_xyz...",
  "status": "active",
  "contextLimit": 10,
  "createdAt": "2025-12-03T10:00:00Z"
}
```
</CodeGroup>

<Tip>
  Save the `secret` from the response — you'll use it to verify webhook signatures and keep your endpoint secure.
</Tip>

### Step 5: Handle agent.message events

When someone texts or calls your number, you'll receive an `agent.message` event with full conversation context:

<CodeGroup>
```json Incoming webhook payload
{
  "event": "agent.message",
  "channel": "sms",
  "agentId": "agent_123",
  "data": {
    "conversationId": "conv_abc",
    "from": "+14155551234",
    "to": "+18571234567",
    "message": "Hey, I need help!",
    "direction": "inbound"
  },
  "conversationState": { "customerName": "John" },
  "recentHistory": [
    { "content": "Previous message", "direction": "inbound", "channel": "sms", "at": "..." }
  ]
}
```
</CodeGroup>

<Note>
  For voice calls, your webhook must return JSON with a `text` field that will be spoken to the caller. See the [Webhooks](/api-reference/webhooks) section for details.
</Note>
